# 🚀 InterviewPrep – Full Stack Project

A modern **Full Stack Web Application** built to help users prepare for interviews with a clean UI and structured backend.

---

## 📌 Features

* 🔐 User Authentication (Login / Signup)
* 📊 Dashboard Interface
* ⚡ Fast and responsive UI
* 🔗 API Integration (Frontend ↔ Backend)
* 🧠 Structured Code Architecture

---

## 🛠️ Tech Stack

### Frontend:

* React.js (Vite)
* HTML, CSS, JavaScript

### Backend:

* Node.js
* Express.js

### Database:

* (MongoDB)

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```
git clone https://github.com/tech-vijay/InterviewPrep.git
```

### 2️⃣ Install dependencies

```
cd backend
npm install

cd ../frontend
npm install
```

### 3️⃣ Run the project

```
# Backend
npm run dev

# Frontend
npm run dev
```
